# 💰 Fluxo Financeiro

Sistema simples para controle de finanças pessoais, permitindo registrar entradas, saídas e acompanhar o saldo de forma prática.

---

## 📌 Sobre o projeto

O **Fluxo Financeiro** é uma aplicação desenvolvida com o objetivo de ajudar no controle financeiro pessoal.

Com ele, é possível:

- Registrar receitas (entradas)
- Registrar despesas (saídas)
- Visualizar saldo atual
- Organizar melhor o dinheiro no dia a dia

---

## 🚀 Tecnologias utilizadas

- React Native
- TypeScript
- Vite
